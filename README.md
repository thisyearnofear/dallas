# Dallas Buyers Club

![cover](.github/cover.png)

This project is a web application inspired by the 1980s-era "Dallas Buyers Club," a real-life operation that distributed unapproved, life-saving treatments to people with AIDS. This fictional marketplace is an homage to that spirit of rebellion and mutual aid.

The site is built with a modern tech stack, including:
*   **Framework**: Preact
*   **Build Tool**: Vite
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS

## Getting Started

To run this project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
